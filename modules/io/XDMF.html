classmethod export_data(network, phases=[], filename='')[source]¶

Saves (transient/steady-state) data from the given objects into the specified file.

Parameters

  • network (OpenPNM Network Object) – The network containing the desired data

  • phases (list of OpenPNM Phase Objects (optional, default is none)) – A list of phase objects whose data are to be included

Notes

This method only saves the data, not any of the pore-scale models or other attributes.
